Uni-Tracker CDN Integration Example

This example demonstrates how to integrate Uni-Tracker using a CDN script.

Integration Steps:

  1. Include the Uni-Tracker script from CDN
  2. Initialize the tracker with your app ID and other configuration
  3. Track events using the UniTracker.track() method

Available Methods:

  • UniTracker.init(config) - Initialize the tracker
  • UniTracker.track(eventName, params) - Track custom events
  • UniTracker.trackPageView() - Track page views (automatic if autoTrackPV is true)
  • UniTracker.setUser(userInfo) - Set user information
  • UniTracker.startSession() - Start a new session
  • UniTracker.endSession() - End the current session
